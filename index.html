プライバシーポリシー

最終更新日：2026年6月24日

本プライバシーポリシーは、Katsura Design Works（以下「開発者」）が提供するアプリ FundaScope および FundaScope Lite（以下「本アプリ」）における 個人情報の取り扱いについて説明するものです。

1. 収集する情報

本アプリが収集・保存する情報は以下のとおりです。

  • J-Quants APIキー：ユーザーが設定画面に入力したAPIキーは、端末内（SharedPreferences）にのみ保存されます。開発者のサーバーには送信されません。
  • 株価・財務データ：J-Quants APIを通じてユーザー自身のAPIキーで取得したデータは、端末内のSQLiteデータベースにローカル保存されます。外部には送信されません。
  • スクリーニング設定・お気に入りリスト：ユーザーが作成したフローチャートやリストは端末内にのみ保存されます。

開発者は、ユーザーの氏名・メールアドレス・位置情報などの個人を特定できる情報を収集しません。

2. 情報の利用目的

収集した情報は、本アプリの機能（株式スクリーニング・指標表示・スコアリング）を提供するためにのみ使用します。

3. 第三者への情報提供

開発者は、ユーザーの情報を第三者に販売・賃貸・共有しません。ただし、以下の第三者サービスを利用しており、それぞれの規約に基づき情報が収集される場合があります。

4. 広告（FundaScope Lite のみ）

無料版 FundaScope Lite は、Google AdMob による広告を表示します。 AdMob はデバイス識別子や広告インタラクション情報を収集することがあります。 詳しくは Google のプライバシーポリシーをご覧ください。

  • Google プライバシーポリシー
  • Google 広告のオプトアウト

5. J-Quants API について

本アプリは、株式会社JPX総研が提供する J-Quants API を利用しています。 本アプリは J-Quants API のデータを配信・再配布するサービスではありません。 データの取得にはユーザー自身が J-Quants（株式会社JPX総研）と別途サービス利用契約を締結し、 ご自身のAPIキーを設定する必要があります。 J-Quants API で取得されるデータの利用は J-Quants API 利用規約 に従い、私的使用の範囲内に限られます。

6. Play Integrity API について

本アプリは Google Play Integrity API を使用して、アプリが正規の環境で実行されているか確認します。 この確認はユーザーのデバイス整合性を検証するために行われ、取得した結果は端末内でのみ使用されます。

7. データの保管と削除

すべてのデータは端末内にのみ保存されます。 アプリをアンインストールすることで、すべてのデータが端末から削除されます。

8. 免責事項

本アプリは投資教育を目的としたツールであり、特定の銘柄への投資を推奨するものではありません。 本アプリの情報に基づく投資判断および損益については、開発者は一切の責任を負いません。 投資はご自身の判断と責任において行ってください。

9. プライバシーポリシーの変更

本ポリシーは必要に応じて更新されることがあります。 変更後のポリシーは本ページに掲載され、掲載時点から有効となります。
